Solution Options

  1. Implement a chain of callback
  2. Save to global scope. For example window.currentState
  3. Move logic to the URL and set the task to the backend developer

Problems of the solutions

  1. The application must remember the order of callbacks
  2. We'll have to update the whole app with ReactDOM.render
  3. Excellent option. But we don't have a backend developer, and the state will be rigidly tied to the URL

React Hooks ü™ù

Hooks are an addition in React 16.8. They let you use the state and do side effects

      const Counter = () => {
        const [count, setCount] = React.useState(0);
        return ( 
          <div> 
            <p>You clicked {count} times</p> 
            <button onClick={() => setCount(count + 1)}> 
              Click me 
            </button> 
          </div> 
        ); 
      }
reactjs.org/hooks-intro

How Hooks Work

  • Hook is a global function. The function is called every time the component is rendered
  • React tracks calls of these functions
  • Hook is a common function but with a bit of magic ü™Ñ

Hooks Limitations

  • The sequence of hook is important. Cannot be used in conditions, loops
  • Hooks can only be used in components or custom hooks
  • No support for all lifecycle methods: getSnapshotBeforeUpdate, getDerivedStateFromError, componentDidCatch

Hooks API ü™ù

  • useState
  • useEffect
  • useCallback
  • useMemo and etc. 
        const Counter = ({ count }) => {
          const [currentCount, setCurrentCount] = React.useState(count);
         
          React.useEffect(() => {
            setCurrentCount(count);
          }, [count]);
         
          return ( 
            <div> 
              <p>You clicked {currentCount} times</p> 
              <button onClick={() => setCount(setCurrentCount + 1)}> 
                Click me 
              </button> 
            </div> 
          ); 
        }

Higher Order Component (HOC)

      const withModalState = (Component) => {
        return class WithModalState extends React.Component {
          constructor(props) {
            super(props);
            this.state = {
              isOpen: false,
            };
            this.handleModalToggle = this.handleModalToggle.bind(this);
          }
       
          handleModalToggle() {
            this.setState({ isOpen: !this.state.isOpen })
          }
       
          render() {
            return <Component {...this.props} onModalToggle={this.handleModalToggle} isOpen={this.state.isOpen} />; 
          }
        }
      }
